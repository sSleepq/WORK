# Основная информация для дипломников

## 1. Запись на руководство

Для записи на дипломное руководство необходимо лично подойти в кабинет  
**с уже выбранной и сформулированной темой дипломного проекта**.

Без выбранной темы запись не производится.

---

## 2. Порядок коммуникации

### Важно

Все вопросы направлять:
- в личные сообщения;
- либо задавать лично (предпочтительно — более быстрый отклик).

### Порядок обработки сообщений

- Ответы предоставляются **в порядке очереди**.
- Если после отправки вопроса вы отправляете дополнительное сообщение с новым вопросом —  
  вы автоматически переноситесь **в конец очереди**.
- Перед отправкой сообщения рекомендуется тщательно сформулировать все вопросы одним сообщением.

---

## 3. Ответственность дипломника

### Важно

Дипломный руководитель **не выполняет дипломную работу за студента**.

Если в процессе выполнения работы возникают трудности:
- поиск решения осуществляется дипломником самостоятельно;
- допускается консультационная поддержка в рамках руководства.

---

## 4. Роль дипломного руководителя

Задачи руководителя:

- Руководство процессом написания дипломного проекта;
- Консультирование по структуре работы;
- Проверка оформления;
- Контроль соответствия установленным требованиям.

---

## 5. Требования к дипломной работе

- Работа оформляется **в соответствии с требованиями ГОСТ**.
- Минимальный объем дипломной работы — **60 страниц** (без учета приложений, если иное не указано методическими рекомендациями).
